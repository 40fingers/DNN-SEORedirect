Regex

To use regex matching in URLs, check the "Use Regex" box on the Edit Mappings page. The incoming URL can now contain a regular expression, e.g.: 

^http[s]?://.*\.40fingers\.net.*$
will match any url on http or https for 40fingers.net or any subdomain.
